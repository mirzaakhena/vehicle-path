  ---                                                                                                                                
  Detail Hooks yang Tidak Terpakai                                                                                                   
                                                                                                                                     
  1. useVehiclePath (hooks/useVehiclePath.ts)                                                                                        
                                                                                                                                     
  - Tujuan: High-level coordinated hook yang menggabungkan useScene, useVehicles, dan useMovement                                    
  - Caller: Hanya unit tests (useVehiclePath.test.ts)                                                                                
  - Catatan: Ini adalah API "batteries-included" untuk consumer. Demo project memilih menggunakan DSL hooks (useSceneDefinition,     
  useInitialMovement, useMovementSequence) sebagai gantinya.                                                                         
                                                                                                                                     
  2. useVehicleEvent (providers/useVehicleEvents.tsx)                                                                                
                                                                                                                                     
  - Tujuan: Convenience hook untuk subscribe ke events via React Context                                                             
  - Caller: Hanya unit tests (useVehicleEvents.test.tsx)                                                                             
  - Catatan: Demo project menggunakan useCreateVehicleEventEmitter langsung dengan eventEmitter.on() daripada hook ini.              
                                                                                                                                     
  ---                                                                                                                                
  Kesimpulan                                                                                                                         
                                                                                                                                     
  Kedua hooks ini bukan dead code dalam arti sebenarnya - mereka adalah alternative public APIs yang disediakan untuk consumer       
  library dengan use case berbeda:                                                                                                   
                                                                                                                                     
  - useVehiclePath: Untuk consumer yang ingin API sederhana tanpa DSL                                                                
  - useVehicleEvent: Untuk consumer yang menggunakan VehicleEventProvider context                                                    
                                                                                                                                     
  Demo project kebetulan tidak menggunakan keduanya karena memilih pendekatan lain (DSL hooks + direct emitter).
